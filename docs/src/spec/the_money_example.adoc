= The Money Example

== 仕様

既存レポート

|====================
|銘柄|株数|価格|合計|
|IBM |1000|25  |25000|
|GE  |400 |100 |40000|
|    |    |総計 |65000|
|====================

レポートを多国籍通貨対応させるために、通貨の情報を加えなければならない。

|====================
|銘柄       |株数  |価格  |合計  |
|IBM       |1000|25 USD  |25000 USD|
|Novartis  |400 |150 CHF |60000 CHF|
|          |    |総計 |65000 USD|
|====================

加えて、為替レートも規定しなければならない

|====================
|換算元|換算先|レート|
|CHF|USD|1.5|
|====================
